# CodeAlpha_Simple_Ecommerce_Store# Simple E-Commerce Store (ShopHorizon)

A minimalist frontend storefront integrated with a lightweight Node.js/Express backend server, built as part of the CodeAlpha Web Development Internship requirements.

## 🚀 Features
* **Dynamic Grid Layout:** Fully responsive catalog displaying curated minimalist products.
* **Interactive Shopping Cart:** Client-side JavaScript DOM manipulation tracks and increments selected items in real-time.
* **Unified UI Shell:** Integrated user login and navigation components.
* **Express Backend:** Structured backend server managing localized application file delivery.

## 🛠️ Built With
* **Frontend:** HTML5, CSS3 (Flexbox & CSS Grid), Vanilla JavaScript (ES6)
* **Backend:** Node.js, Express.js framework

## 💻 How to Run Locally
1. Clone or download this project folder.
2. Open your terminal inside the project directory.
3. Install the required backend dependencies:
   ```bash
   npm install
   node server.js
